Introduction

This tool converts edX student_state_*.csv to a format that is useful for manually analyzing data.

How to use?

Select multiple student_state_ files on 'Add student_state files...' Choose file dialog

Select mapping file if you want to use one.

Enter prefix used that prefixes mapping file keys to match csv file format.

Run Start conversion to get results right in the browser, and generate table below the form.

When you are happy with the form, press Select Table, copy and paste the table in Excel or Google Docs. or Download to download CSV file.

File format and prefix

Mapping file should be CSV file with two columns: question_id, text_to_map_to, for example:

            f8d12bf3717f4a9484f200ea1e3d9f14_2_1,My 1st question
            f8d12bf3717f4a9484f200ea1e3d9f14_3_1,My 2nd question

Prefix should be rest value that prefixes this key, for example: i4x-MITx-MyCOURSE-problem-
